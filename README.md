# Akira - Premium Portfolio & CMS

A high-end, luxury-themed portfolio website for **Akira**, built with Next.js 15, Tailwind CSS, and Supabase.

## ✨ Key Features

- **Obsidian Design System**: A bespoke luxury aesthetic with glassmorphism, cinematic animations, and high-fidelity typography.
- **Full Admin Dashboard**:
  - **Booking Calendar**: Visual management of client sessions.
  - **Portfolio CMS**: Mosaic bento-grid management with media uploads.
  - **Blog Engine**: Dynamic content creation and management.
  - **CRM**: Client relationship management and testimonial publishing.
- **Performance Optimized**: Built with React 19, Server Components, and optimized media loading.
- **PWA Support**: Installable as a native-like application on all devices.
- **SEO Ready**: Automatic metadata generation and semantic HTML structure.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org) (App Router)
- **Styling**: Vanilla CSS + Tailwind-ready utility architecture
- **Animations**: Framer Motion
- **Database/Auth**: Supabase
- **Deployment**: Vercel

## 🛠️ Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result

## 🔒 Security

- Sensitive keys are managed via `.env.local` and are excluded from version control.
- Administrative pages are protected by Supabase Auth and strict RLS (Row Level Security) policies.
